# nodejs-courses
CLI app for note taking. Basic CRUD app for note taking, stores/updates everything within db.json.

Example of commands:

##Creating a new note:
`note new "clean room" --tags "important"`
##List all notes:
`note all`
##Finding a specific note:
`note find "clean"`
##Removing a note:
`note remove "<id of note>"`
##Removing all notes:
`note clean`
